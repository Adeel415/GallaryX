# 📱 Smart Gallery App

A modern Flutter-based smart media gallery application that allows users to view, manage, and secure photos and videos stored on their device. The app is built with a clean architecture and optimized for performance.

---

## ✨ Features

- View all photos from device storage
- Browse and play videos smoothly
- Smart media filtering (Images & Videos)
- Secure Vault for private media storage
- Favorite media support
- Sorting (Date, Name, Size)
- Fast loading with pagination
- Clean and responsive UI
